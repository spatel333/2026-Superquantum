# iQuHackathon (iQuack) 2026
This is my work on Superquantum's problem set at the 2026 MIT Quantum Hackathon.

These problems focus primarily on optimizing the T-count of real world quantum circuits.

T-counts are relevant because they are a valuable metric for the cost of any given quantum circuit.
Most Quantum circuits leverage gates which exist in theory but not in practice <cite>.
The task for any given quantum computational researcher is to convert theoretical gates into practical ones which we can implement (Clifford + T Gates).
Optimizations comes into play when adressing the number of T-Counts to use, since these are the more expensive operations.

The Superquantum problem set seeks to replicate this work by showing a series of puzzles which leverage linear algebreic equations to further reduce the number of T Gates (T-count).
